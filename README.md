# Portfolio

A personal portfolio website showcasing projects, skills, and experience.

## Overview

This is a responsive portfolio website built with HTML, CSS, and JavaScript. It features an interactive particle animation effect and is styled with Bootstrap for a modern, professional appearance.

## Features

- **Responsive Design** - Mobile-friendly layout that adapts to all screen sizes
- **Interactive Particles** - Animated particle effect for visual appeal
- **Bootstrap Framework** - Utilizes Bootstrap for consistent styling and components
- **Bootstrap Icons** - Icon library for visual enhancements
- **Professional Layout** - Clean, organized structure for showcasing work and skills

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- No build tools or server required for local viewing

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Aleguiojo777/Portfolio.git
```

2. Navigate to the project directory:
```bash
cd Portfolio
```

3. Open `index.html` in your web browser or serve using a local web server:
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js http-server
npx http-server
```

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactivity and animations
- **Bootstrap 5** - Responsive framework
- **Bootstrap Icons** - Icon library

## Customization

### Updating Content

Edit `index.html` to add your own content, projects, and information.

### Styling

Modify `css/style.css` to customize colors, fonts, and layouts to match your brand.

### Animations

Adjust the particle effect settings in `js/particles.js` to control animation behavior.
